CP2: Research

Key Words

  • Color management

    : : the controlled conversion between the color representations of various devices.

  • Preflight

    : The Preflight tool analyzes the contents of a PDF to determine its validity for print production .

  • Prepress:

    :the term used in the printing and publishing industries for the processes and procedures that occur between the creation of a print layout and the final printing.

  • Classic Curve

    : AKA the Bezier Curve; A mathematical function that provides for the generation of curves suitable for outlining graphics or characters of any typeface, style or siz.

  • Dot Gain:

    the effect of halftone dots growing in area between the original film and the printed sheet.

  • Image end points

    : A feature of a communications system that enables one or more stations to check with other systems to see if there is a message ready to be sent.

  • Halftone placement:

    : Placement of dots for one color channel in view of dot placement of one or more other color channels can enhance image quality.

Key Words Cont.

  • Gray balance

    : The neutral shades of gray that are produced in color printing when cyan, magenta and yellow are combined..

  • Gang run printing

    : To print two or more finished products on the same sheet during one press run.

  • Photoshop: color modes:

    : 8-bit color/grayscale, In 8-bit color mode, the color monitor uses 8 bits for each pixel, making it possible to display 2 to the 8th power (256) different colors or shades of gray.

  • Photoshop: curves

    :Bezier Curve

  • InDesign: crop vs. reframing

    (Crop an object by making its frame smaller, reframe an object by moving the object to fit in the frame.

  • InDesign: swapping grayscale image color

    : Changing grey in a photo to specific colors.

  • InDesign: frame color vs. object color

    : the frame color refers to the color of the frame around the picture, the object color is the color of the picture.
